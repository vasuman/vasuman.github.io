Box, Hop! Box

Play

“Box, Hop! Box” is a game where the objective is to keep climbing while avoiding the edges of the screen and falling boxes.

I originally made this for js13k for which the theme was reversal. Hence, the flipping controller. I messed up the final submission and the result is that it only works in browsers that support a few ES6 features. The version on this site works fine though.

It was mainly designed to be played on touchscreen devices.
